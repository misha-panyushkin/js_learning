Логирование - поиск ошибок:

  • alert
  • console.log
  • try {..} catch () {..}
  • и др.

Вызываем функцию: alert ("Show me!");

Любые функции:

  • $ -> $("h2")
  • $(document).ready -> $(document).ready(..)
  • $("h1").remove -> $("h1").remove()
  • $("h1").css -> $("h1").css("opacity", .5)
  • и др.

Моя первая функция

        function SayMyName () { 
          // Частная собственность!
        }
    
        При обращении к имени SayMyName, будет выведено:
        function SayMyName () {...}

Складываем слова в предложения:

        function SayMyName (name) {
          var str = name;
          str = "Hello" + name + "!";
          alert (str);  // Вызываем функцию внутри другой функции 
        }

Круговое движение

Циклы

        Возможность повторять однотипные действия:
        for(var i = 0; i < 100; i++) {
          ...
        }
    
        В обратном порядке:
        for(var i = 100; i > 0; i--) {
          ...
        }

Аналог в jQuery:

        .each();  // Да, это тоже функция!
    
        Используем для коллекции:
        $("h2").each(function (..) {..});
    
        Количство элементов в коллекции == количеству вызовов.
        function (i, e) { .. }
        i - индекс, e - элемент

Атака массивов

        "цветок", "черный дипломат", "очки", "плащ"
        "леон", "матильда", "стетфилд", "фикус"
        1, 2, 3, 4
        Всё это напоминает очередь
        Так и есть!
    
        Беспорядочный список:
        "цветок", "черный дипломат", "очки", "плащ"
    
        Упорядочный список:
        "подъём", "упрожнения на пресс", "в магазин за молоком",
 "завтрак"

Совершенно неважно!

        Окаймляем в скобки:
        ["подъём", "упрожнения на пресс", "в магазин за молоком",
 "завтрак"];
    
        Записываем в переменную:
        var morning = ["подъём", "упрожнения на пресс", "в магазин за молоком",
 "завтрак"];
    
        Утренний распорядок:
        var morning = ["подъём", "упрожнения на пресс", "в магазин за молоком", "завтрак"];
    
        morning[0] // -> "подъём"
        morning[1] // -> "упрожнения на пресс"
        morning[2] // -> "в магазин за молоком"
        morning[3] // -> "завтрак"
